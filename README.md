# Neural Junkie — CAD pack

Official domain pack for [Neural Junkie](https://github.com/camronwood/neural-junkie).

Install via desktop **Settings → Domain packs → Pack store**, or sideload `dist/cad-<version>.zip`.

## Develop

```bash
make verify
make pack-zip   # dist/cad-<version>.zip
```

Tag `v1.0.0` and push to publish the release zip to GitHub.
